# Object detection using YOLO

Detect objects using prebuilt YOLO model

## Usage
```
$ uv run detectobject.py --help
usage: detectobject.py [-h] [--input-video INPUT_VIDEO] [--output-video OUTPUT_VIDEO] [--temp-video TEMP_VIDEO]

Run object detection and store it in a video file.

options:
  -h, --help            show this help message and exit
  --input-video, -i INPUT_VIDEO
                        Path to the input video file
  --output-video, -o OUTPUT_VIDEO
                        Path to save the processed output video
  --temp-video, -t TEMP_VIDEO
                        Store a temporary video for further processing
```
